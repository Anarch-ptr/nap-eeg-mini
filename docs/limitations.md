## Covariance Intervention Evidence Limitation

The first covariance-separation intervention uses only one frozen matched
LOW/HIGH pair per subject. It therefore remains vulnerable to matched-pair
realization effects and unmeasured trial-level differences. A stable
directional result is stronger than observational correlation, but is not
definitive causal proof and must be labelled **First Intervention-Style
Evidence**.

Independent matched-pair replication is required before deep micro-mechanism
interpretation. Pair Set 2 must be selected deterministically from the already
frozen candidate bank, without model-performance information, and frozen in a
separate commit before replication training. Even successful replication must
be labelled **Replicated Intervention-Supported Evidence**, not definitive
neurophysiological causality.

The evidence hierarchy is:

1. **ROBUST_CANDIDATE_SIGNAL**: stable observational training-data-only association.
2. **FIRST_INTERVENTION_STYLE_EVIDENCE**: directional effect from one frozen matched pair per subject.
3. **REPLICATED_INTERVENTION_SUPPORTED_EVIDENCE**: the same directional effect independently reproduced with a second preregistered pair realization.
4. **MECHANISM-SUPPORTED EVIDENCE**: a separately tested measurable process explains how the manipulated property changes model behavior.

These levels must not be collapsed. None alone automatically establishes a
universal neurophysiological causal mechanism.
