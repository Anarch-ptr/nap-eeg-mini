## Covariance Intervention Evidence Limitation

The first covariance-separation intervention used one frozen matched LOW/HIGH
pair per subject. Its preregistered result was
`HETEROGENEOUS_OR_WEAK_INTERVENTION_EFFECT`: HIGH performed worse for four
subjects, better for four, and tied for one. The median HIGH-minus-LOW effect
was 0.00 percentage points; seed-level median directions were positive for
seeds 42 and 43 and negative for seed 44. The completed experiment should
therefore be described as a **First Intervention-Style Test With a
Heterogeneous/Weak Effect**, not as causal confirmation.

The result remains vulnerable to matched-pair realization effects and
unmeasured trial-level differences even though run composition, acquisition
order, class counts, and covariance within-class dispersion were matched. The
sample contains only nine subjects, which also limits population-level
generalization.

The earlier `ROBUST_CANDIDATE_SIGNAL` remains valid as a cross-subject
observational association. It must not be interpreted as a universal
manipulable mechanism: a between-subject association does not imply that
within-subject manipulation will have the same direction or magnitude.

Pair Set 2 replication was conditional on Pair Set 1 producing a stable
`HIGH_SEPARATION_WORSE` or `HIGH_SEPARATION_BETTER` classification. Pair Set 1
produced neither, so Pair Set 2 remains unselected and untrained. Selecting it
after this outcome would be an outcome-driven continuation. No
micro-mechanism analysis was performed, and NAP was not implemented because
the mechanistic evidence remained insufficient.

The evidence hierarchy remains conceptually useful—observational signal,
directional first intervention, independent replication, and measurable
mechanism—but the present evidence stops at a heterogeneous first
intervention-style test. These levels must not be collapsed, and none alone
establishes universal neurophysiological causality.
